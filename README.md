# ToDoodle Pixel Quest 🏰⚔️🐉

ToDoodle Pixel Quest is a gamified to-do app where everyday tasks become epic adventures.  
You manage **quests** (projects or goals) and **tasks** (steps to complete them), while a progress bar shows a little knight moving toward the castle.

---

## 🌟 Concept
- **Quests** = projects, goals, or big challenges.
- **Tasks** = steps to complete along the way.
- Progress bar shows your knight's journey to the castle.

---

## ✨ Current Progress (MVP)
- Able to create, view, edit, and delete quests.
- Add tasks to quests.
- Track task completion.

> ⚠️ **Work in progress:** Features like pixel art visuals, achievements, and user accounts are planned for future updates.

---

## 🛠️ Built With
- Ruby on Rails
- TailwindCSS
- PostgreSQL

---

## ⚡ Getting Started

### Prerequisites
- Ruby 3.x
- Rails 7.x
- PostgreSQL

### Setup
```bash
git clone https://github.com/YOURUSERNAME/todoodle-pixel-quest.git
cd todoodle-pixel-quest
bundle install
bin/rails db:create db:migrate
bin/dev
Then open http://localhost:3000 to try it out (features are in progress).
```
